# Mediafy

A Next.js SaaS application for uploading and managing videos and images using Cloudinary, Clerk authentication, Prisma, and PostgreSQL.

## Features

- Secure authentication and user management with Clerk
- Image upload and editing support
- Predefined image cropping presets for popular social media platforms
- Video upload and compression using Cloudinary
- Browse and manage recent uploads from the home page
- PostgreSQL database integration with Prisma ORM
- Dockerized application setup for consistent deployment
- CI/CD pipeline with GitHub Actions

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Prisma ORM
- PostgreSQL
- Clerk Authentication
- Cloudinary
- Docker
- GitHub Actions
- DaisyUI
